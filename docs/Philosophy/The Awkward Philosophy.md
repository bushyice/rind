##       -- Prelude --


I watch myself write yet another rust code from an empty file, as I only imagine but how it will shape out- It didn't take long before I was playing in a virtual machine, testing what init systems really are. 

It wasn't long before I imagined a new system, a new software, something I could potentially introduce as a philosophy.

Behold- The awkward system, one that lives by contradiction, yet serves both sides as a trade for the other.

[[Rind]] is a project that speaks a language of its own, a language of conflict and [[Balance|balance]]. And you, my friend, are about to delve into a thread of concepts that fight in harmony.
